# NIR: Разработка метода оптимизации системы тендерных продаж

Данный репозиторий содержит материалы научно-исследовательской работы, посвящённой разработке метода оптимизации ценообразования в системе электронных тендерных продаж на основе вероятностного моделирования и методов оптимизации.

---

## Структура репозитория

- **`EDA.ipynb`**  
  Первичный анализ данных (Exploratory Data Analysis), включающий:
  - обзор и очистку исходного датасета;
  - анализ распределений цен;
  - расчёт базовых статистических показателей;
  - выявление ключевых закономерностей, используемых на последующих этапах исследования.

---

## Цель работы

Разработка и обоснование метода оптимального выбора цены заявки в электронных тендерах, направленного на максимизацию ожидаемой прибыли компании с учётом:
- неопределённости рыночной цены;
- регуляторных ограничений (НМЦК, минимальная маржа и др.).

---

## Ключевые направления исследования

1. Анализ исторических данных электронных торгов.  
2. Вероятностное моделирование распределения рыночной цены лота.  
3. Формирование оптимизационной модели выбора цены заявки.  
4. Экспериментальная оценка эффективности предложенного метода.
